This plugin allows forms to be filled via the querystring, in the format [UFC field type]-[field ID] = [value];
Values need to be in an URL-encoded format.<> Example: ?ufc-txt-edit-title=My Article Title
Current Fields:
ufc-txt Text Fields: Title, Summary, block, contributor- NOT Body Text!
ufc-chk Check Box [TRUE/FALSE]
ufc-sel Select
ufc-msl Multiselect- Only ADDS to selection, manual removal if wrong
ufc-bod Body Text
ufc-fll [download page] opens first file to media library; filters on value
ufc-mdc [media browser list]UFC-Media Check- check the number of items expected against number of media found, closes tab if match found
THE FOLLOWING OCCUR AFTER FIELDS
ufc-sav-pub-cls [TRUE]save, publish, and close tab
ufc-submit-cls [TRUE]click submit and close tab

Youll have to see the results in the console.
